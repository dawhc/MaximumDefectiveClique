# Maximum Defective Clique

Maximum defective clique algorithm 

## Build
```bash
cmake . && make
```


## Run
- Command
```bash
bin/run -d <dataset> -k <k> -a <algorithm>
```
> Available algoritms: MDC / RussianDoll / KDBB

- Example
```bash
bin/run -d datas/socfb-Harvard1 -k 1 -a MDC
```
